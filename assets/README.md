# Placeholder Assets

This directory should contain the following image files:

- `icon.png` (1024x1024) - App icon
- `splash.png` (2048x2048) - Splash screen
- `adaptive-icon.png` (1024x1024) - Android adaptive icon
- `favicon.png` (48x48) - Web favicon

## Creating Assets

See [docs/ASSETS.md](../docs/ASSETS.md) for detailed instructions on creating these assets.

## Quick Setup

For development, you can use placeholder images:

1. **Option 1**: Use online generators
   - Visit https://appicon.co/
   - Upload a bicycle image
   - Download generated assets

2. **Option 2**: Create simple colored squares
   - Use any image editor
   - Create solid color images with text
   - Save in PNG format

3. **Option 3**: Copy from template
   - Search "expo app icon template"
   - Customize with your branding

## Note

The app will still run without custom assets, but Expo will show warnings. Custom assets are required for production builds.
